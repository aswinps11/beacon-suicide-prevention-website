# Beacon

A modern, responsive web application built with HTML, CSS, and Bootstrap to provide support and resources for mental health awareness and crisis intervention.

## Features

- **Modern UI Design**: Built with Bootstrap 5 and custom CSS for a sleek, professional appearance
- **Responsive Layout**: Fully responsive design that works seamlessly across all devices
- **Navigation System**: Fixed navigation bar with smooth backdrop blur effects
- **Resource Hub**: Dedicated resources page with organized support materials
- **Beautiful Styling**: Custom gradients, typography with Poppins font family, and smooth interactions

## Project Structure

```
.
├── project.html      # Main landing page
├── resource.html     # Resources and support page
├── README.md         # This file
└── css/
    └── bootstrap.css # Bootstrap stylesheet
```

## Pages

### Main Page (`project.html`)
- Hero section with full-height banner
- Navigation bar with links to resources
- Multiple content sections with gradient overlays

### Resources Page (`resource.html`)
- Sidebar navigation for easy resource browsing
- Organized support and crisis intervention resources
- Fixed navigation for accessibility

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with gradients, backdrop filters, and animations
- **Bootstrap 5**: Responsive grid system and components
- **Google Fonts**: Poppins font family for modern typography

## Setup & Usage

1. Clone or download this repository
2. Open `project.html` in your web browser
3. Navigate to the Resources page for support materials

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Getting Started

No installation required! Simply open the HTML files in your browser to view the project.
